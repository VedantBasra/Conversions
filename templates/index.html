Technical Documentation

1. convert_office_to_pdf()

  • Purpose: Convert DOCX, PPTX, or XLSX files to PDF.
  • Libraries: LibreOffice (headless mode).
  • Input: File path to a supported Office format.
  • Output: PDF file path or None on failure.
  • Edge Cases: Requires LibreOffice installed; may fail silently for corrupted files.

2. convert_images_to_pdf()

  • Purpose: Merge multiple images into a single PDF.
  • Libraries: Pillow (PIL).
  • Input: List of image file paths.
  • Output: Combined PDF file path.
  • Edge Cases: Converts non-RGB automatically; empty image list returns None.

3. convert_pdf_to_images()

  • Purpose: Render each PDF page as an image.
  • Libraries: PyMuPDF (fitz).
  • Input: PDF file path, output folder, base filename.
  • Output: List of image paths and a ZIP archive path.
  • Edge Cases: Encrypted or massive PDFs may fail or lag.

4. extract_images_from_pdf()

  • Purpose: Extract embedded raster images from PDFs.
  • Libraries: PyMuPDF (fitz).
  • Input: PDF file path, output directory, base name.
  • Output: List of image paths and a ZIP archive path.
  • Edge Cases: Vector images are ignored; CMYK auto-converted.

5. convert_markdown_to_pdf()

  • Purpose: Render a styled Markdown file to PDF.
  • Libraries: markdown, pdfkit, wkhtmltopdf.
  • Input: Path to a .md file.
  • Output: Generated PDF file path.
  • Edge Cases: Broken Markdown renders poorly.

6. convert_html_to_pdf()

  • Purpose: Convert .html file to PDF.
  • Libraries: pdfkit, wkhtmltopdf.
  • Input: HTML file path.
  • Output: PDF output file path.
  • Edge Cases: JavaScript may not render; requires enable-local-file-access.

7. convert_image_format()

  • Purpose: Reformat an image file to a new type (JPG, PNG, WEBP).
  • Libraries: Pillow.
  • Input: Input image path, output path, desired format.
  • Output: Reformatted image path.
  • Edge Cases: Non-RGB images auto-convert; invalid formats throw errors.

8. resize_image()

  • Purpose: Resize an image via fixed dimensions or a scale factor.
  • Libraries: Pillow.
  • Input: Image path, width/height/scale (at least one required).
  • Output: Path to resized image.
  • Edge Cases: Invalid input types or extreme scales cause errors.
